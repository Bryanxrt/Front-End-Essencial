## 📝 Exercícios

### Exercício 01 – Apresentação Pessoal
Página HTML com uma breve apresentação pessoal contendo:
- Título principal com nome
- Parágrafo de apresentação
- Lista de hobbies e interesses
- Imagem representativa
- Link para rede social

### Exercício 02 – Card de Produto
Card de produto para loja virtual contendo:
- Imagem do produto
- Nome do produto em destaque
- Descrição curta
- Preço com valor promocional
- Botão de comprar

### Exercício 03 – Layout com Flexbox
Reprodução de um layout utilizando Flexbox com:
- Header
- Sidebar lateral
- Área de conteúdo principal
- Footer
